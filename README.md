# Lucid-Text
A simple text-editor desktop application built using the [PyGTK](https://en.wikipedia.org/wiki/PyGTK) software. 
<br><br>
This app supports common text-editor features such as 
- text format options
- alignment options
- text wrapping
- search & mark, and
- find & replace, etc.
